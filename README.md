这是一个blener中的相机列表插件（进行了简单的标题汉化，以及修改了对4.2系统中的渲染器切换的代码修改）

基本功能：

1、快捷键alt+c,可以快速显示场景中的相机列表

2、可以快速以当前视图创建相机

3、可以快速设置相机视野和渲染方式
